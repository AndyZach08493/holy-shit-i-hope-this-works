Merges two sorted array segments theArray[first..mid] and theArray[mid+1..last] into one sorted array.

Precondition:
first <= mid <= last. The subarrays theArray[first..mid] and theArray[mid+1..last] are each sorted in increasing order.
Postcondition:
theArray[first..last] is sorted.
Parameters:
theArray The given array.
first The beginning of the first segment in theArray.
mid The end of the first segement in theArray. mid + 1 marks the beginning of the second segment.
last The last element in the second segment in theArray.
Note:
This function merges the two subarrays into a temporary array and copies the result into the original array theArray.
